Restaurant Menu Display

A simple React application that displays a restaurant menu using reusable components, props, and conditional rendering. Built using Vite for fast development and modern tooling.

Components
1. MenuItem Component

Displays a single menu item

Accepts props like name, price, description, and isVeg

Shows a “VEG” badge only if the dish is vegetarian

Designed as a reusable component used throughout the menu list

2. MenuList Component

Renders a list of MenuItem components

Demonstrates the use of .map() to dynamically generate UI from data

Run Locally

Clone the repository:

git clone <https://github.com/mohammedajwadh/restaurant-menu.git>


Navigate into the project directory:

cd restaurantmenu_project


Install dependencies:

npm install


Start development server:

npm run dev
